# 4.0.Functions[Dart]

- A function is a block of organized, reusable code that is used to perform a single, related action.
- Creating the Function

```ruby
void functionName(){
//To Do Something
}
```

- Calling the Function

```ruby
functionName();
```
- Named Function

```ruby
void functionName(){
//To Do Something
}
```

- Anonymous Function

```ruby
(){
//To Do Something
}
```

- Example in Dart using Anonymous Function,

```ruby
onPressed: (){
//To Do Something
}
```
