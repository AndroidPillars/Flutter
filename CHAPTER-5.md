# List[Dart]

__Example 1[Getting Name]__
```ruby
void main() {
  
  List<String> myList = [
    'List1',
    'List2',
    'List3',
    'List4'
  ];
  
  print(myList[0]);
    
}
```

__Example 2[Getting Index Value]__
```ruby
void main() {
  
  List<String> myList = [
    'List1',
    'List2',
    'List3',
    'List4'
  ];
  
  print(myList.indexOf('List1'));
    
}
```
